# Elevator-Dispatching
MFC基于对话框应用程序-电梯调度系统

完成语言：C++

需要软件：Microsoft Visual Studio

主要功能：
模拟真实场景电梯使用（两个电梯）

技术难点：
两个电梯之间的多线程协调

运行方式：
下载后直接双击HotelSystem.sln即可运行
